# Movie Mosaic

A premium, highly-polished movie discovery web application built with React, Vite, and Tailwind CSS. Explore popular movies, search for your favorites, discover by genre, and watch trailers right from the app.

## ✨ Features

- **Discover & Search**: Browse trending movies or search for specific titles.
- **Config-Driven Genres**: Dynamic routing for any genre defined in the configuration.
- **Movie Details**: View comprehensive movie details including cast, similar movies, budget, revenue, and more.
- **Infinite Scrolling**: Seamlessly browse through hundreds of movies without clicking "Next Page".
- **Trailer Integration**: Watch high-quality YouTube trailers in an embedded, accessible modal.
- **Favorites System**: Save movies to your personal favorites list with instant UI feedback (toasts).
- **Responsive & Accessible**: Fully responsive design with a mobile hamburger menu. Keyboard navigable dropdowns and full screen-reader support.
- **Optimized Caching**: Powered by TanStack React Query for lightning-fast background refetching, caching, and robust error handling.
- **Premium UI/UX**: Shimmering skeleton loaders, vibrant hover effects, glassmorphism elements, and smooth transitions.

## 🚀 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Routing**: React Router v7
- **Styling**: Tailwind CSS
- **Data Fetching & Caching**: TanStack React Query
- **Icons & Polish**: Custom CSS animations and dynamic typography

## 🛠️ Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ISHIMWE-Prince-Arnaud/Movie-Mosaic.git
   cd Movie-Mosaic
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add your TMDB API key:
   ```env
   VITE_TMDB_API_KEY=your_tmdb_v3_api_key_here
   ```
   > You can get a free API key by creating an account at [The Movie Database (TMDB)](https://www.themoviedb.org/).

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```
